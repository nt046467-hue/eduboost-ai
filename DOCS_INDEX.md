## ❓ FAQ

### Where do I get my API key?

→ https://aistudio.google.com/apikey

### Where do I put my API key?

→ In `.env.local` file (locally) and Netlify Dashboard (production)

### How long does it take to deploy?

→ About 15 minutes total from start to live

### What if the build fails?

→ Check `NETLIFY_CHECKLIST.md` troubleshooting section

### How do I get a custom domain?

→ After first deploy: Netlify Settings → Domain Management

### Can I go back to a previous version?

→ Yes! Netlify allows one-click rollbacks

### Does my API key stay secret?

→ Yes! It's in `.env.local` (gitignored) and Netlify Dashboard only

### What happens when I git push?

→ Netlify auto-builds and deploys automatically (CI/CD)
